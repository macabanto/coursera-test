The browser uses its selector API to traverse the DOM(Document Object Model), and pick out the elements matching the selector(id/class name)
CSS Rules can be applied by:

  • element selectors. all elements are affected
    p{
    color:blue;
    }
  • class selectors. element must have class name. referred to in CSS Rule with "."
    can affect different types of elements so long as they share the attribute being modified
    CSS: .classname{
    color:red;
    }
    HTML: <p class="classname">
  • id selectors. ELEMENT(only one element can have this id). referred to in CSS Rule with "#"
    since id's are unique per page, only one element is affected.
    CSS: #idname{
    color:yellow;
    }
    HTML: <div id="idname">

Selectors can be grouped within a CSS Rule. Separated by comma.
